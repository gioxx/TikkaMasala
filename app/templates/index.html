How to use Tikka Masala

Use this section only when you need the setup steps or want to verify the token first.

Before creating a backup, make sure you have your Cloudflare Account ID, the Tunnel ID you want to work with, and an API token with the required permissions. The app uses the token for the current request and can prefill it later if you configured persistence.

To find the Account ID, open the Cloudflare dashboard, go to the Account home page, then use Copy account ID from the account menu. You can also find it in the API section of the account overview.

To create the API token, open My Profile → API Tokens for a user token or Manage Account → API Tokens for an account-owned token (you can also use this direct link), then create a custom token with:

  • Account: Cloudflare Tunnel - Edit
  • Zone: DNS - Edit

For backup-only usage, you can reduce the token to read-only permissions later.

To find the Tunnel ID, go to Zero Trust → Networks → Connectors → Cloudflare Tunnels, open your tunnel, and copy the ID from the details page. If you prefer, use Load tunnels below after entering your Account ID and API token (recommended).
